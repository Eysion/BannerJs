参数：
*options的默认属性说明
*loading  		//加载banner模块前的展示内容<必须是 标签类型 && class && id>
*bannerBox		//包含轮播块的模型
*bannerList 	//轮播的块状模型
*bannerItem 	//单个轮播元素的模型
*bannerShow		//一次展示的元素个数
*gonext		    //加载下一张图片的元素
*goprev		    //加载上一张图片的元素
*goDisk 		  //焦点模块模型
*ItemPadding	//单个轮播元素间的补白
*goDiskWidth 	//单个焦点元素的宽度
*bannerDisk 	//是否开启焦点
*bannerwheel	//鼠标滚轮事件
*bannerauto		//是否开启自动轮播
*autoTime		  //自动轮播的时间

欢迎试用并提出修改意见。
